Purpose

The 53L7A1 BSP is a software component intenteded to be used within the STM32Cube ecosystem. This software implements the BSP v2.0 specifications for the X-NUCLEO-53L7A1 Expansion Board on STM32. It is built on top of STM32Cube software technology that ease portability across different STM32 micro-controllers.

Here is the list of references:

  • X-NUCLEO-53L7A1: 8x8 multizone with wide field of view ranging sensor expansion board based on VL53L7CX for STM32 Nucleo
  • VL53L7CX: Time-of-Flight 8x8 multizone ranging sensor with wide field of view
  • VL53L7CX-SATEL: Breakout board for VL53L7CX
